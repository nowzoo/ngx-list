NgxListColumnValueFn

NgxListColumnValueFn: function
  • Defined in lib/api.ts:43

A function that, given a record and a column key, returns the value of the key for purposes of comparison, i.e. for sorting and filtering records.

The function may return undefined for any column key. In that case we fall back to:

  • for any value that is not a string, the raw value
  • the strings, the lowercased value
